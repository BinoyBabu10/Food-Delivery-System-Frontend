# 🍔 Food Delivery System — Frontend

A modern, responsive food delivery web application built with **React**, **Vite**, and **Tailwind CSS**. This frontend provides a seamless interface for browsing restaurants, ordering food, and managing deliveries.

---

## 🚀 Tech Stack

| Technology | Purpose |
|---|---|
| [React](https://react.dev/) | UI component library |
| [Vite](https://vitejs.dev/) | Build tool & dev server |
| [Tailwind CSS](https://tailwindcss.com/) | Utility-first styling |
| [ESLint](https://eslint.org/) | Code linting |
| JavaScript (ES6+) | Primary language |

---

## ✨ Features

- 🏠 Landing page with food categories and featured restaurants
- 🍽️ Browse menu items and restaurant listings
- 🛒 Shopping cart management
- 📦 Order placement and tracking
- 🔐 User authentication (Login / Register)
- 📱 Fully responsive design for mobile and desktop

## 🛠️ Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/BinoyBabu10/Food-Delivery-System-Frontend.git
   cd Food-Delivery-System-Frontend
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   ```

4. **Open your browser** and navigate to `http://localhost:5173`
